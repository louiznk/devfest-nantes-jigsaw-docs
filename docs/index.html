# Hand's on Jigsaw

# The begining

Vous sortez de chez (l')Oracle qui vous a annoncé un futur bien troublant... Vous allez évoluer vers une nouvelle vie, une nouvelle version de vous-même, le nouvel élu, vous serez...

le nouveau super-héro Java JIGSAW

Dôté des pouvoirs de modularisation au build ET au runtime, vous apporterez les packages privés aux ops et aux devs.

Êtes-vous prêt à vouloir comprendre l'origine de vos pouvoirs et à les utiliser pour le bien de tous ?

Si tel est le cas, rendez-vous au chapitre suivant, où votre aventure commencera...
